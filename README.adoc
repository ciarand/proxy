proxy
=====

[quote]
A development proxy with logging and redirect-rewriting

Installation
------------

[source,bash]
----
go get -u github.com/ciarand/proxy
----

Usage
-----

[source,bash]
----
# start the proxy in one shell:
proxy -from=https://www.google.com -to=http://0.0.0.0:8080

# and in another, run curl:
curl localhost:8080 | wc -l # rearranged for clarity:

# result from proxy shell:
INFO[0040] request started                               client_address=[::1]:58732 method=GET request=/
INFO[0040] request complete                              elapsed=534.38243ms status=200

# result from curl shell:
% Total    % Received % Xferd  Average Speed   Time     Time     Time  Current
                               Dload   Upload  Total    Spent    Left  Speed

8
----
